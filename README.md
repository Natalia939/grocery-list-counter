# Grocery List Counter

A simple Python script that tracks grocery items, counts duplicates, and displays a sorted list. Ideal for learning Python dictionaries, input handling, and basic data organization.

## Features

- Accepts multiple grocery items from the user.
- Automatically counts repeated items.
- Displays the final list in **alphabetical order**.
- Prints counts with item names in uppercase.
- Handles end-of-input gracefully using Ctrl+D (Linux/Mac) or Ctrl+Z (Windows).

## Requirements

- Python 3.x
